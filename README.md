# xc - eXeCute project tasks from a readme file

## Installation

```
go install github.com/joe-davidson1802/xc/cmd/xc@latest
```

## Tasks

__get__: get dependencies of the project
```
go get ./...
```

### Tests

__test__: test project
```
go test ./...
```

__testshort__: run short tests
```
go test ./... -short
```
